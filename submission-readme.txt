ISSM CA3 Readme

1. File List

    CA3ModelV1_model_array_trained\
        - Trained autoencoder array models for testing
        - Used by ISSM_CA3_GC_Array_Test.py
    
    CA3ModelV1_model_array\
        - Autoencoder array models generated by ISSM_CA3_GC_Array_Model_Training.py
        
    data\temp_data.csv
        - Raw dataset 
        
    CA3ModelV1_model_trained.hdf5
        - Trained Conv3D model for testing
        - Used by ISSM_CA3_GC_Test.py
    
    CA3ModelV1_threshold.hdf5
        - Conv3D model generated by ISSM_CA3_GC_Training.py
    
    CA3ModelV1_threshold_trained.hdf5
        - Trained Conv3D model threshold for abnormal status
    
    CA3ModelV1_dataset.hdf5
        - Pre-precessed training and testing dateset
        
    CA3ModelV1_model.hdf5
        - Conv3D model generated by ISSM_CA3_GC_Training.py
    
    ISSM_CA3_GC_Array_Model_Training.py
        - Training application for autoencoder array model
        
    ISSM_CA3_GC_Array_Test.py
        - Testing application for autoencoder array model
     
    ISSM_CA3_GC_Training.py
        - Training application for Conv3D model
    
    ISSM_CA3_GC_Test.py
        - Testing application for Conv3D model
        
2. Run Conv3D model

2-1 Training Conv3D model
    - Run ISSM_CA3_GC_Training.py
    
2-1 Testing Conv3D model
    - Run ISSM_CA3_GC_Test.py

3. Run Autoencoder Array model

2-1 Training Autoencoder Array model
    - Run ISSM_CA3_GC_Array_Model_Training.py
    
2-1 Testing Autoencoder Array model
    - Run ISSM_CA3_GC_Array_Test.py
